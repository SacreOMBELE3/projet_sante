# Présentation du projet

Ce projet de Santé Publique est le travail de S2 et S3 .

Son objectif est de mettre en pratique les notions fondamentales du langage Python acquises au cours de la formation, notamment :

- les variables ;
- les constantes ;
- les commentaires ;
- les chaînes de caractères formatées (f-strings) ;
- les opérateurs arithmétiques et de comparaison ;
- les structures conditionnelles (if,elif,else) ;
- les calculs d'indicateurs de santé.
